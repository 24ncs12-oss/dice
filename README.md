# Dicee

A fun and simple web-based dice game where two players compete to see who rolls the higher number!

## Description

Dicee is a classic dice rolling game implemented as a web application. When you refresh the page, both players roll their dice, and the winner is announced. It's perfect for quick decision-making or just for fun!

## Features

- Random dice rolls for two players
- Visual dice images (d1.png to d6.png)
- Animated gradient background
- Automatic winner determination
- Responsive design

## How to Play

1. Open `index.html` in your web browser
2. Refresh the page to roll the dice
3. See who wins or if it's a draw!

## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript (ES6)

## Project Structure

## Installation

No installation required! Simply clone the repository and open `index.html` in any modern web browser.

```bash
git clone <your-repo-url>
cd dice
# Open index.html in your browser
```

## Browser Support

Works in all modern web browsers that support HTML5, CSS3, and JavaScript.
